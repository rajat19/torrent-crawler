## Torrent Crawler and Search
> Search for torrents using command line
```bash
pip3 install search-torrent
search-torrent
```

### Contributing
```bash
git clone https://github.com/rajat19/torrent-search.git
cd torrent-search
pip install -r requirements.txt
python setup.py install
python search.py
```
